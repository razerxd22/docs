## Artistic Effects

<CardGroup cols={2}>
  <Card title="Half Invert" icon="circle-half-stroke">
    `filter half_invert` (*attachment) - Invert half of the image
  </Card>
  <Card title="Neon" icon="lightbulb">
    `filter neon` (*attachment) - Add neon glow effects
  </Card>
  <Card title="LSD" icon="rainbow">
    `filter lsd` (*attachment) - Apply a psychedelic color effect
  </Card>
  <Card title="Matrix" icon="binary">
    `filter matrix` (*attachment) - Apply a Matrix-style effect
  </Card>
</CardGroup>

## Pixel Effects

<CardGroup cols={2}>
  <Card title="Minecraft" icon="cube">
    `filter minecraft` (*attachment) - Convert image into Minecraft blocks
  </Card>
  <Card title="Emojify" icon="face-smile">
    `filter emojify` (*attachment) - Convert image into emoji pixels
  </Card>
  <Card title="Letters" icon="font">
    `filter letters` (*attachment) - Convert image into ASCII letters
  </Card>
  <Card title="Bayer" icon="chess-board">
    `filter bayer` (*attachment) - Apply a Bayer matrix dithering effect
  </Card>
</CardGroup>
