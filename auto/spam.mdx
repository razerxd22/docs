# Enabling the filter

This will block users from spamming in the chat

Subcommand: `.automod spam enable` <br />
Help: Enable the automod spam filter. <br />


<Tabs>
    <Tab title="Usage">
       ```
       Example: .automod spam enable
       ```
    </Tab>
</Tabs>

## Change the timeout minutes for the filter

Subcommand: `.automod spam timeout` <br />
Help: Change the timeout duration for automod. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .automod spam timeout (minutes)
       Example: .automod spam timeout 5
       ```
    </Tab>
</Tabs>

# Ignore channels, roles and users from the filter

## Ignoing a target

Subcommand: `.automod spam ignore add` <br />
Help: Ignore a target to the automod spam filter. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .automod spam ignore add (target)
       Example: .automod spam ignore add @encrypted
       ```
    </Tab>
</Tabs>

## Remvoing a target from ignored list

Subcommand: `.automod spam ignore remove` <br />
Help: Remove a target from the automod spam filter. <br />


<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .automod spam ignore remove (target)
       Example: .automod spam ignore remove @encrypted
       ```
    </Tab>
</Tabs>

## Viewing the ignored users

Subcommand: `.automod spam ignore list` <br />
Help: View all the ignored targets <br />

<Tabs>
    <Tab title="Usage">
       ```
       Example: .automod spam ignore list
       ```
    </Tab>
</Tabs>
