# Enabling the filter

This will block users from sending all types of links in the server

Subcommand: `.automod links enable` <br />
Help: Enable the automod links filter. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Example: .automod links enable
       ```
    </Tab>
</Tabs>

## Change the timeout minutes for the filter

Subcommand: `.automod links timeout` <br />
Help: Change the timeout duration for automod. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .automod invites timeout (minutes)
       Example: .automod invites timeout 5
       ```
    </Tab>
</Tabs>

# Ignore channels, roles and users from the filter

## Ignoing a target

Subcommand: `.automod links ignore add` <br />
Help: Ignore a target to the automod links filter. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .automod links ignore add (target)
       Example: .automod links ignore add @encrypted
       ```
    </Tab>
</Tabs>

## Remvoing a target from ignored list

Subcommand: `.automod links ignore remove` <br />
Help: Remove a target from the automod links filter. <br />


<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .automod links ignore remove (target)
       Example: .automod links ignore remove @encrypted
       ```
    </Tab>
</Tabs>

## Viewing the ignored users

Subcommand: `.automod links ignore list` <br />
Help: View all the ignored targets <br />

<Tabs>
    <Tab title="Usage">
       ```
       Example: .automod invites ignore list
       ```
    </Tab>
</Tabs>
