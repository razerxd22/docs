# Enable Word Filter

This will block users from sending saved words in the server


Subcommand: `.automod words enable` <br />
Help: Enable the automod words filter. <br />


<Tabs>
    <Tab title="Usage">
       ```
       Example: .automod words enable
       ```
    </Tab>
</Tabs>

## Blacklist a word

Subcommand: `.automod words add` <br />
Help: Add a word to the blacklist. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .automod words add (word)
       Example: .automod words add zecroyn
       ```
    </Tab>
</Tabs>

# Change the target of the auto moderation words

<Info>
There are two available targets: `timeout` & `delete` , On target timeout it will delete the messages too
</Info>


<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .automod words target (target)
       Example: .automod words target timeout
       ```
    </Tab>
</Tabs>


# Ignore channels, roles and users from the filter

## Ignoing a target

Subcommand: `.automod words ignore add` <br />
Help: Ignore a target to the automod words filter. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .automod words ignore add (target)
       Example: .automod words ignore add @encrypted
       ```
    </Tab>
</Tabs>

## Remvoing a target from ignored list

Subcommand: `.automod words ignore remove` <br />
Help: Remove a target from the automod words filter. <br />


<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .automod words ignore remove (target)
       Example: .automod words ignore remove @encrypted
       ```
    </Tab>
</Tabs>

## Viewing the ignored users

Subcommand: `.automod words ignore list` <br />
Help: View all the ignored targets <br />

<Tabs>
    <Tab title="Usage">
       ```
       Example: .automod words ignore list
       ```
    </Tab>
</Tabs>
