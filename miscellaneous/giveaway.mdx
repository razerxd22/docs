# Start a Giveaway

Subcommand: `.giveaway start` <br />
Help: You can start a giveaway using this command. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .giveaway start (duration) (winners) (prize)
       Example: .giveaway start 14h 1 Nitro
       ```
    </Tab>
</Tabs>

# Cancel a giveaway

You can cancel a giveaway with the `giveaway cancel` command.

<Warning>
This will not choose any winners and will delete the giveaway message
</Warning>

Subcommand: `.giveaway cancel` <br />
Help: You can cancel a giveaway using this command. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .giveaway cancel (message_id)
       Example: .giveaway cancel 1386641528918900747
       ```
    </Tab>
</Tabs>
