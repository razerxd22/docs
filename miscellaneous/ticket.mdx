# Category 
Setup a category where all the tickets will be created.

Subcommand: `.ticket category` <br />
Help: Set the ticket category where tickets for that topic will be created. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .ticket category (topic) (category_id)
       Example: .ticket category Support 1370275085998358629
       ```
    </Tab>
</Tabs>


# Support Role
Select a support role for the ticket system. This role can close tickets, view them and add/remove members from the ticket.

Subcommand: `.ticket support` <br />
Help: Set the ticket support role for the topic. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .ticket support (topic) (role)
       Example: .ticket support Support @TicketsManager
       ```
    </Tab>
</Tabs>


# Ticket Topics
Create multiple ticket topics to have a better overview of your tickets.

Subcommand: `.ticket topic add` <br />
Help: Add ticket topics. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .ticket topic add (name) [emoji]
       Example: .ticket topic add Support :Support: 
       ```
    </Tab>
</Tabs>

<Info>
You can also edit existing topics.
</Info>

```
?ticket topic name <topic> <name>
?ticket topic emoji <topic> <emoj>
```

# Ticket Create embed
To setup a ticket create embed use `ticket setup` command.


Subcommand: `.ticket setup` <br />
Help: Setup a ticket creation embed. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Example: .ticket setup 
       ```
    </Tab>
</Tabs>

If you don't want the default ticketing embed, you can create one yourself. Go to [Embed Builder](https://nyxie.space/embed).

After you have finished Customizing your embed, click on "Generate Code" button and copy it with "Copy Code" button

Subcommand: `.ticket setup` <br />
Help: Setup a ticket creation embed. <br />

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .ticket setup (embed_code)
       Example: .ticket setup {embed}$v{color: #57F287}$v{title: Ticket setup}$v{description: Create a ticket using the button below}
       ```
    </Tab>
</Tabs>
