Reward users for representing your server vanity in their status.

Subcommand: `vanity channel` \
Help: Set the channel that vanity messages are sent to.

<div id="channel" />

<Tabs>
  <Tab title="Usage">
    ```
    Syntax: .vanity channel (channel)
    Example: .vanity channel #vanity
    ```
  </Tab>
</Tabs>

<br />

Subcommand: `vanity role` \
Help: Change or set the role that is given upon adverising.

<div id="role" />

<Tabs>
  <Tab title="Usage">
    ```
    Syntax: .vanity role (role)
    Example: .vanity role @supporters
    ```
  </Tab>
</Tabs>

<br />

Subcommand: `vanity set` \
Help: Set the vanity that the bot will log.

<div id="role" />

<Tabs>
  <Tab title="Usage">
    ```
    Syntax: .vanity set (vanity)
    Example: .vanity set .gg/cypherbot
    ```
  </Tab>
</Tabs>
