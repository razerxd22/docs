## Creating a welcome channel

<Steps>
  <Step title="Head over to your Discord Server">
    Click on the Discord Server that you would like to setup welcome for using Cypher.
  </Step>
  <Step title="Create the channel">
    Setup permissions for the channel with the desired name, for this example we
    will be using `#welcome`
  </Step>
</Steps>

## Creating an embed and setting the welcome channel

<Steps>
  <Step title="Creating the welcome embed">
    Go to the [Embed builder](https://cypherbot.space/embed) And create a custom embed for your welcome setup for more information regarding variables:
    [/resources/placeholders](/resources/placeholders) and for emebds go to: [/resources/embeds](/resources/embeds)
  </Step>
  <Step title="Setup the welcome channel">
    Run the `.welcome channel` command which will give you further information, for this
    example we'll run: `.welcome channel #welcome` this is well setup the welcome channel next we will run the `.welcome message`. command which will setup the custom welcome message example we'll run `.welcome message {embed}$v{description: Welcome to {server.name}}`
  </Step>
</Steps>
