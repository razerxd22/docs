## Setting up loggings

Subcommand: `loggings setup` <br /> Help: Create & set all the logging events

<div id="enable"/>
<Tabs>
    <Tab title="Usage">
       ```
       Example: .loggings setup
       ```
    </Tab>
</Tabs>
<br />

## Adding a logging event

Subcommand: `loggings add` <br /> 
Help: Add specific logging channels for specific events <br /> 

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .loggings add (event) (channel)
       Example: .loggings add mod #mod-logs
       ```
    </Tab>
</Tabs>

## Removing a logging event

Subcommand: `loggings remove` <br /> 
Help: Remove specific logging event <br /> 

<Tabs>
    <Tab title="Usage">
       ```
       Syntax: .loggings remove (event)
       Example: .loggings remove mod
       ```
    </Tab>
</Tabs>

## Viewing logging configuration

Subcommand: `loggings config` <br /> 
Help: View configuration of loggings in the server<br /> 
<div id="config"/>
<Tabs>
    <Tab title="Usage">
       ```
       Example: .loggings config
       ```
    </Tab>
