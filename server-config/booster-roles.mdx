## Base role

<Steps>
  <Step title="Head over to your Discord Server">
    Click on the Discord Server that you would like to setup Booster roles for using
    Evict.
  </Step>
  <Step title="Create the base role">
  Use `.boosterrole base (role)` To Setup the base position for booster roles. For example `.boosterrole base @boosters` 
  <br />
  </Step>
</Steps>

## Creating booster role
<Steps>
<Step title="Create the Booster Role">
  Run `boosterrole create (name) (color)` to create a booster role for yourself and your friends example: `boosterrole create EncryptedsRole #ff00ff`, After setting up the base role, go to your commands channel and use `.boosterrole (color)`. For example, `.boosterrole color #fff` to set a new color. To rename your booster role, use `.boosterrole rename (name)`.
  <br />
</Step>
</Steps>
